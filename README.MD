# Fullmoon
2D即时战斗游戏的Demo
## Features
 - 采用OOP的C++编写. 由于使用了非开源图像, 只上传了源码部分, 源码若未注明作者则为本人.
 - 主要技术: 
    - GDI+
    - Win32
    - ELSE
 - 主要算法:
    - 动态精灵切割
    - 四叉树
    - 卡马克卷轴*
    - 寻路*
 - 主要实现内容:(('*'号表示只实现了局部或实现得不完善)
    - 2D图形处理
        - 精灵
        - 瓦片地图
    - 人物8方向移动及其动画
    - 物理碰撞处理
    - 战斗伤害
    - 伤害特效
    - 事件
        - 鼠标点击
        - 物品拾取
        - 地图阻挡
        - 传送*
        - 经验*
    - 物品使用*
    - AI*
